title: Faster, less noisy docs search
date: '2026-04-25'
category: Docs
benefit: Search now matches page titles only — no more wading through paragraph snippets to find what you need.
---

Press `⌘K` and start typing. You'll see a clean list of pages instead of a wall of half-matched sentences.

This is how docs search should feel: you type a topic, you get the page. We rebuilt the search index to surface page titles only, and re-skinned the results so the page you want is always the first thing you see.

Try `tracking`, `commission`, or `payout` and you should land on the right page in one keystroke.
