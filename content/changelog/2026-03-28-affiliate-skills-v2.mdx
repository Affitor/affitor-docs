If you use Claude Code or another agent framework, the [affiliate-skills](https://github.com/Affitor/agent-skills) library now ships 47 ready-to-use skills.

A few you might reach for first:

- `validate-tracking` — checks that an advertiser's pixel and webhook are firing correctly
- `partner-outreach` — drafts a personalised invite for a creator using their public profile
- `commission-calculator` — works out partner payouts under your current rules
- `program-setup` — walks an advertiser through a full program configuration

The library is open source and works against the public Affitor API.
