## Support Resources

- **[System Status](/support/status/)** — See the current status-page availability and what to do if you suspect an issue
- **[FAQ](/faq/)** — Common questions about pricing, tracking, payouts, and setup

## Before Contacting Support

To get the fastest help:

1. Search the docs for the flow you're implementing
2. Review the relevant tracking guide or troubleshooting section
3. Check [System Status](/support/status/) to confirm whether a public status page is available yet
4. Include your program ID, affected flow, and any relevant screenshots or request payloads when you contact support
