# Quickstart Guide

Get your affiliate program up and running in minutes with our comprehensive quickstart guide.

## Step-by-Step Setup

Follow these guides in order to set up your affiliate program:

1. **[Create Account](/advertisers/quickstart/create-account/)** - Set up your advertiser account on Affitor
2. **[Setup Program](/advertisers/quickstart/setup-program/)** - Configure your affiliate program settings
3. **[Partner Approval & Quality Control](/advertisers/quickstart/partner-approval-quality-control/)** - Set up partner approval workflows
4. **[View Performance](/advertisers/quickstart/view-performance/)** - Access your performance dashboard
5. **[Define Commission](/advertisers/quickstart/define-commission/)** - Set up your commission structure
6. **[Commission Approval & Cash Flow](/advertisers/quickstart/commission-approval-cash-flow/)** - Manage commission approvals
7. **[Payouts](/advertisers/quickstart/payouts/)** - Configure affiliate payouts

## What You'll Achieve

By completing this quickstart guide, you'll have:

- ✅ A fully configured affiliate program
- ✅ Commission structure in place
- ✅ Partner approval process set up
- ✅ Payout system configured
- ✅ Performance tracking enabled
