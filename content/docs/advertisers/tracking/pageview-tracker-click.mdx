The Pageview Tracker captures when visitors arrive via affiliate links. This is Step 1 of 3 in Affitor tracking (Click → Lead → Sale) and the foundation that all other tracking depends on.

**Prerequisite:** An active Affitor program with a Program ID (found in **Settings → Integration**).

---

## What It Does

When someone clicks a partner's affiliate link (`https://yoursite.com?ref=PARTNER123`), the tracker:

1. Detects the `?ref=` parameter
2. Requests Affitor backend for a unique `customer_code`
3. Stores `customer_code` in a cookie (60-day lifetime)
4. Creates customer-partner relationship in database
5. Sends a click event to Affitor

All subsequent events (signups, purchases) are linked via `customer_code` and `user_id`.

---

## Installation

Add this script tag to your website's `<head>` section:

```html
<script
  src="https://api.affitor.com/js/affitor-tracker.js"
  data-affitor-program-id="YOUR_PROGRAM_ID"
  data-affitor-debug="false">
</script>
```

:::tip[Program ID]
When you copy code from your Affitor dashboard (**Settings → Tracking**), your Program ID is pre-filled. If copying from these docs, replace `YOUR_PROGRAM_ID` with your ID from Settings.
:::

That's it. The script handles everything automatically.

---

## Where to Add the Script

Add the script to **every page** where affiliate traffic might land.

**Recommended:** Add to your site's global `<head>` so it loads on all pages.

## Framework Examples

### HTML

```html
<!DOCTYPE html>
<html>
<head>
  <title>Your Site</title>
  <script
    src="https://api.affitor.com/js/affitor-tracker.js"
    data-affitor-program-id="YOUR_PROGRAM_ID"
    data-affitor-debug="false">
  </script>
</head>
<body>
  <!-- Your content -->
</body>
</html>
```

### Next.js (Pages Router)

```tsx
// pages/_document.tsx
import { Html, Head, Main, NextScript } from 'next/document'

export default function Document() {
  return (
    <Html>
      <Head>
        <script
          src="https://api.affitor.com/js/affitor-tracker.js"
          data-affitor-program-id="YOUR_PROGRAM_ID"
          data-affitor-debug="false"
        ></script>
      </Head>
      <body>
        <Main />
        <NextScript />
      </body>
    </Html>
  )
}
```

### Next.js (App Router)

```tsx
// app/layout.tsx
import Script from 'next/script'

export default function RootLayout({ children }) {
  return (
    <html>
      <body>
        {children}
        <Script
          src="https://api.affitor.com/js/affitor-tracker.js"
          data-affitor-program-id="YOUR_PROGRAM_ID"
          data-affitor-debug="false"
          strategy="afterInteractive"
        />
      </body>
    </html>
  )
}
```

:::caution[App Router: Script Placement]
The `<Script>` component with `strategy="afterInteractive"` must be inside `<body>`, not `<head>`. It is a React component that needs to be in the component tree to function correctly. Placing it in `<head>` causes hydration errors or the script not loading.
:::

## Configuration

| Attribute | Required | Description |
|-----------|----------|-------------|
| `src` | Yes | Affitor tracker script URL |
| `data-affitor-program-id` | Yes | Your Program ID (from **Settings → Integration**) |
| `data-affitor-debug` | No | Set `"true"` for test mode (default: `"false"`) |

## How Affiliate Links Work

Partners receive links in this format:

```
https://yoursite.com?ref=PARTNER123
```

Or with a landing page:

```
https://yoursite.com/pricing?ref=PARTNER123
```

The tracker detects any URL with `?ref=` and captures the partner code.

## Verifying Installation

### Quick Test

Visit your site with a test affiliate parameter to verify the full flow:

```
https://yoursite.com?ref=TEST123
```

Open Developer Tools (F12) → **Application** → **Cookies** and confirm `customer_code` is set.

### Test Mode

Enable debug mode to see tracking events in your browser console:

```html
<script
  src="https://api.affitor.com/js/affitor-tracker.js"
  data-affitor-program-id="YOUR_PROGRAM_ID"
  data-affitor-debug="true">
</script>
```

**Then:**

1. Open your browser's Developer Tools (F12)
2. Go to the **Console** tab
3. Visit your site with `?ref=TEST123`

## Troubleshooting

### Script Not Loading

**Symptoms:** No cookies set, no console messages in debug mode

**Check:**
- Script URL is correct
- No Content Security Policy blocking the script
- Script is in `<head>`, not `<body>` (except Next.js App Router which uses `<body>`)

### Cookies Not Set

**Symptoms:** Script loads but cookies are missing

**Check:**
- Site is served over **HTTPS** (required for secure cookies)
- No browser extensions blocking cookies

### Events Not Appearing in Dashboard

**Symptoms:** Cookies set but no events in Affitor

**Check:**
- Program ID is correct
- Check browser **Network** tab for failed requests
