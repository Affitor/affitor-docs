Get affiliate tracking live in three steps. Each step builds on the previous one: clicks → signups → sales.

---

## Step 1 — Install the Script

Add the Affitor script to every page of your site. It auto-detects affiliate links (`?aff=` parameter) and stores the `click_id` in a cookie for later attribution.

```html
<script
  src="https://api.affitor.com/js/affitor-tracker.js"
  data-affitor-program-id="YOUR_PROGRAM_ID"
  data-affitor-debug="false">
</script>
```

Replace `YOUR_PROGRAM_ID` with your program ID from the dashboard.

**What this does:**

- Detects `?aff=` in the URL
- Stores a `click_id` in a first-party cookie (60-day window)
- Fires a `pageview` event to Affitor — partners can see their click counts

**Verify:** Open your browser console. You should see:

```
[Affitor] Tracker loaded — program: YOUR_PROGRAM_ID
[Affitor] Attribution stored — click_id: cust_...
```

---

## Step 2 — Track Signups

After a user successfully registers, call `signup()`. This links the signup to the affiliate who referred them.

```html
<script>
// Call this after your signup/registration succeeds
await window.affitor.signup('user_123', 'user@example.com');
</script>
```

**Example with a form:**

```html
<script>
document.getElementById('signup-form').addEventListener('submit', async function(e) {
  e.preventDefault();

  var email = document.getElementById('email').value;

  // Run your own signup logic first, then on success:
  await window.affitor.signup(currentUser.id, email);
});
</script>
```

:::note
`customerKey` (first argument) is required. It lets Affitor match the signup to a later payment even if the user checks out with a different browser or device.
:::

**What this does:**

- Validates the stored `affitor_click_id` cookie
- Creates a lead record linked to the referring partner
- Partners can now see: Clicks → **Signups**

---

## Step 3 — Track Sales

When a customer pays, Affitor needs to know. Choose the method that fits your checkout:

### Stripe Checkout

Pass Affitor metadata in your Stripe session. Affitor's webhook listener handles the rest automatically.

```js
// Server-side — Node.js example
const session = await stripe.checkout.sessions.create({
  line_items: [{ price: 'price_xxx', quantity: 1 }],
  mode: 'payment',
  success_url: 'https://yoursite.com/success',
  cancel_url:  'https://yoursite.com/cancel',
  metadata: {
    affitor_click_id: 'cust_xyz789',      // from cookie: affitor_click_id
    affitor_customer_key: 'user_123',
    program_id:    'YOUR_PROGRAM_ID',
  },
});
```

Read the cookie values server-side or pass them from the frontend via a hidden form field.

**What this does:**

- Matches the payment to the correct partner via `click_id`
- Calculates commission based on your program tier
- Partners can now see: Clicks → Signups → **Sales**

---

## You're Done

Once all three steps fire successfully, your integration is complete. You can verify it end-to-end using the [Testing Integration](/advertisers/tracking/testing-integration/) guide.

| Step | Event | Dashboard view |
|------|-------|---------------|
| Script installed | `pageview` | Clicks |
| Signup tracked | `lead` | Signups |
| Payment tracked | `conversion` | Revenue |
