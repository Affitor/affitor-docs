Get affiliate tracking live in three steps. Each step builds on the previous one: clicks → signups → sales.

---

## Step 1 — Install the Script

Add the Affitor script to every page of your site. It auto-detects affiliate links (`?ref=` parameter) and stores the `customer_code` in a cookie for later attribution.

```html
<script>
  (function(w, d, s) {
    var f = d.getElementsByTagName(s)[0],
        j = d.createElement(s);
    j.async = true;
    j.src = 'https://cdn.affitor.com/tracker.js';
    j.setAttribute('data-program-id', 'YOUR_PROGRAM_ID');
    f.parentNode.insertBefore(j, f);
  })(window, document, 'script');
</script>
```

Replace `YOUR_PROGRAM_ID` with your program ID from the dashboard.

**What this does:**

- Detects `?ref=PARTNER_CODE` in the URL
- Stores a `customer_code` in a first-party cookie (60-day window)
- Fires a `pageview` event to Affitor — partners can see their click counts

**Verify:** Open your browser console. You should see:

```
[Affitor] Tracker loaded — program: YOUR_PROGRAM_ID
[Affitor] Attribution stored — customer_code: cust_...
```

## Step 2 — Track Signups

After a user successfully registers, call `trackLead()`. This links the signup to the affiliate who referred them.

```html
<script>
// Call this after your signup/registration succeeds
window.affitor.trackLead({
  email: 'user@example.com',   // Required: user's email
  name: 'John Doe',            // Optional: full name
  user_id: 'user_123',        // Recommended: your internal user ID
  additional_data: {
    signup_method: 'email',    // Optional: how they signed up
  }
});
</script>
```

**Example with a form:**

```html
<script>
document.getElementById('signup-form').addEventListener('submit', function(e) {
  e.preventDefault();

  var email = document.getElementById('email').value;
  var name  = document.getElementById('name').value;

  // Run your own signup logic first, then on success:
  window.affitor.trackLead({
    email: email,
    name: name,
    user_id: currentUser.id,   // Pass your internal ID here
    additional_data: {
      signup_method: 'form',
    }
  });
});
</script>
```

:::note
`user_id` is strongly recommended. It lets Affitor match the signup to a later payment even if the user checks out with a different browser or device.
:::

**What this does:**

- Validates the stored `customer_code` cookie
- Creates a lead record linked to the referring partner
- Partners can now see: Clicks → **Signups**

## Step 3 — Track Sales

When a customer pays, Affitor needs to know. Choose the method that fits your checkout:

### Option A — Stripe Checkout (Recommended)

Pass Affitor metadata in your Stripe session. Affitor's webhook listener handles the rest automatically.

```js
// Server-side — Node.js example
const session = await stripe.checkout.sessions.create({
  line_items: [{ price: 'price_xxx', quantity: 1 }],
  mode: 'payment',
  success_url: 'https://yoursite.com/success',
  cancel_url:  'https://yoursite.com/cancel',
  metadata: {
    partner_code:  'PARTNER_ABC123',   // from cookie: affitor_partner_code
    customer_code: 'cust_xyz789',      // from cookie: affitor_customer_code
    program_id:    'YOUR_PROGRAM_ID',
  },
});
```

Read the cookie values server-side or pass them from the frontend via a hidden form field.

### Option B — Manual `trackConversion()`

Use this if you are not using Stripe Checkout, or want to record sales from your own payment processor.

```html
<script>
// Call this after payment confirmation
window.affitor.trackConversion({
  amount: 99.00,           // Required: sale amount
  currency: 'USD',         // Optional: defaults to USD
  order_id: 'ORDER_123',  // Optional: your order reference
  additional_data: {
    product_name: 'Premium Plan',
  }
});
</script>
```

**What this does:**

- Matches the payment to the correct partner via `customer_code`
- Calculates commission based on your program tier
- Partners can now see: Clicks → Signups → **Sales**

## You're Done

Once all three steps fire successfully, your integration is complete. You can verify it end-to-end using the [Testing Integration](/advertisers/tracking/testing-integration/) guide.

| Step | Event | Dashboard view |
|------|-------|---------------|
| Script installed | `pageview` | Clicks |
| Signup tracked | `lead` | Signups |
| Payment tracked | `conversion` | Revenue |
