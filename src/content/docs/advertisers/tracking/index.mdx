## Tracking Events

Affitor tracks three main event types:

1. **Click** - When a user clicks an affiliate link
2. **Signup** - When a user creates an account
3. **Payment** - When a user makes a payment (via Stripe)

## Integration Requirements

To fully integrate Affitor tracking, you'll need:

- Access to your website's codebase
- Stripe account (for payment tracking)
- Ability to add JavaScript to your pages
- Ability to make server-side API calls (for signup tracking)
